
## Краткое назначение основных файлов

| Файл / папка               | Назначение                                                                 | Когда изменять |
|----------------------------|-------------------------------------------------------------------------------|----------------|
| `index.html`               | Точка входа. Содержит только фон, логотип, меню, копирайт и контейнер для контента | Редко          |
| `css/variables.css`        | Глобальные цвета, радиусы, glass-эффекты, шрифты                               | При смене бренд-цветов |
| `css/background.css`       | Фон всего сайта (градиент, текстура, vignette и т.д.)                        | При смене фона |
| `css/style.css`            | Базовые стили, .glass-apple, hover, анимации, .element и т.д.                | Часто          |
| `css/menu.css`             | Бургер-меню и полное меню                                                    | При изменении меню |
| `css/config-*.css`         | Позиционирование, размеры, отступы конкретно для одной страницы              | При изменении раскладки страницы |
| `js/main.js`               | Навигация, fetch модулей, меню, dark mode, parallax, свайпы                  | При добавлении новых страниц |
| `js/contact.js`            | Обработка отправки формы (submit, индикатор загрузки, сообщения)             | При изменении формы |
| `modules/*.html`           | Содержимое каждой страницы (подгружается динамически)                        | При изменении контента страницы |

## Правила работы со структурой

1. Новая страница = новый файл в `modules/` + новый config-файл в `css/` + 1 строка в `js/main.js` (в объекте `names`)
2. Никогда не используй инлайн-стили (`style="..."`) в модулях — только классы и CSS-переменные
3. Все пути к css в модулях — относительные (`../css/config-....css`)
4. После добавления новой страницы — обнови максимальное значение в свайпах в main.js (`currentPage < N`)

## Как добавить новую страницу (пример: AE-Space)

1. Создать `modules/space.html`
2. Создать `css/config-space.css`
3. В `js/main.js` добавить в `names`: `7: 'space'`
4. В меню добавить пункт: `<li><a href="#" onclick="navigate(7); toggleMenu()">AE-Space</a></li>`
5. В карточках проектов добавить `onclick="navigate(7)"` на нужную карточку

Готово.
